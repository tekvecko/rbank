# User Interface (Tailwind CSS)

UI vychází z moderních mobilních bankovních aplikací s důrazem na čistý a elegantní **Dark Mode** design (navy blue / dark šedá).

## Principy a Technologie
- **Tailwind CSS:** Používá se pro veškeré stylování aplikací přímo v className atributech (utility-first přístup).
- **Konzistentní paleta:** Pozadí `#22252e`, panely `#2c2f38`, akcentní barvy (modrá `#3b82f6`, žlutá `#fcd535`).
- **Responzivita:** Ačkoliv je aplikace primárně mobilní, UI prvky musí správně využívat Flexbox (`flex`, `justify-between`) pro korektní zobrazení na různých velikostech displejů.

## Hlavní obrazovky
- Dashboard (Přehled účtů)
- Historie (Výpis transakcí)
- Platby a Produkty
- Profil uživatele
