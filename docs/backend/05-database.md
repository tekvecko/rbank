# Backend Database

Databáze je jediným zdrojem pravdy (Single Source of Truth).

Backend je jediná vrstva, která do databáze čte a zapisuje. Frontend operuje pouze nad daty, která mu backend poskytne přes API.

## Cílový stav dokumentace
Budoucí technická dokumentace k databázi bude obsahovat:
- Schéma (ERD diagram)
- Definice tabulek a relací
- Strategie indexování pro rychlé vyhledávání
- Systém migrací (verzování databázových změn)
